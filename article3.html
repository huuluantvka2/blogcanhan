HTML && CSS

Những thứ cần biết để giỏi hơn trong FrontEnd

Tìm nguồn học tốt và phù hợp với bản thân: vì mỗi người một cách học khác nhau, có thể cách học của họ ổn với họ nhưng không ổn với bạn, có người sẽ thích học bằng cách đọc tài liệu, nhưng cũng có người tiếp thu tốt hơn khi xem video, những lời khuyên từ người khác thì nên tham khảo chứ đừng nghe hay áp dụng theo răm rắp phải có tư duy và lập trường riêng của bản thân.

Học phân tích từ Design ra cấu trúc HTML nhanh nếu HTML không phân tích cấu trúc tốt thì code CSS sẽ khó khăn

HTML thì có thể học ở w3schools , nếu cảm thấy không hợp đọc lại điều 1, tìm nguồn khác như Youtube hay các khoá học online (khuyến khích tự học là chính)

HTML thì học về các thẻ, cách đặt tên class, cách sử dụng các thẻ semantic, ví dụ không thể sử dụng thẻ <h1> để bọc một đoạn văn chỉ vì muốn font chữ của nó to lên. Tìm hiểu sự khác nhau giữa thẻ inline và thẻ block, cách sử dụng thẻ đúng đắn, các attribute hay gặp của chúng như input thì có type, placeholder, có pseudo là :focus, thẻ a thì là :hover, :active...

Nên tìm hiểu thêm về BEM, style guide để học cách đặt tên, Atomic design, các thư viện CSS để xem cách ngta tổ chức code, các preprocessor như Sass để hỗ trợ code nhanh hơn. Học cách Google, Stack Overflow để tự giải quyết vấn đề.

Học tiếng anh (ý này khá quan trọng). Mình là một học sinh khối A và đã rất vất vã để lấy lại căn bản, tiếng anh thực sự rất quan trọng với ngành IT nói riêng và tất cả các ngành khác nói chung. Ngành này nếu các bạn tìm tài liệu tiếng việt để đọc thì đa phần là công nghệ nó đã cũ.

Tìm hiểu về git, github hoặc các dịch vụ lưu trữ tương tự để biết cách quản lý source code khi làm dự án với team, xử lý commit, pull, push, rebase, pull request, tạo issue...

Tìm hiểu về UI UX để biết đẹp xấu, trải nghiệm người dùng, xem nhiều các UIs đẹp ở dribbble, behance, medium muz li, để mở mang màu sắc, con mắt về thẩm mỹ

Học CSS cũng thế tìm nguồn học chất lượng như w3schools, MDN, Youtube, Udemy, Codecademy, Freecodecamp, Scrimba, ....

Học flexbox quan trọng để dàn layout có thể học tại flexbox.io của wesbos free, tạo tài khoản bằng email miễn phí rồi học, ở đây có dạy luôn css grid cũng free nếu muốn học thêm để mở mang kiến thức. Nên master các thuộc tính như position, flexbox, before, after, transition, transform, display, các pseudo cơ bản như hover, focus, .... Nếu giỏi flexbox thì việc chia layout rất khoẻ, position giúp làm menu, menu đa cấp, nhiều thứ khác, transition để làm mọi thứ mượt mà hơn, before hay after làm nhiều thứ kinh khủng, transform giúp thay đổi element và nhiều thuộc tính css quan trọng khác mà các bạn cần biết thêm.

Khi học CSS thì chú ý từng thuộc tính một nếu bạn muốn pro thì phải hiểu cách nó hoạt động, như tại sao dùng flexbox thì các cột lại cao bằng nhau, tại sao lại tràn ra ngoài, tại sao bị bể layout, tại sao code không chạy đúng thứ tự, tại sao nó ưu tiên hơn, tại sao biến không hoạt đông... và nên tự đặt nhiều câu hỏi tại sao như vậy để hiểu sâu về nó. Còn nếu học loa qua thì không hiểu hết thì làm đôi khi lỗi chả biết tại sao....(mình cũng thường xuyên bị lỗi =((( )

Thực hành nhiều là điều cần thiết kể khi bạn đã khá giỏi, vì kiến thức là bao la. Mình làm rất nhiều ở máy tính cũng như trên Codepen. Vậy tìm giao diện ở đâu để tập cắt. Google không thiếu chẳng qua các bạn lười tìm, ví dụ muốn cắt giao diện thì google "PSD template free" ra mấy trang như freebiebugs hay sysm đầy template ra tha hồ tải về mà cắt.

Nên biết các trang code online như Codepen để có vài thứ code online sẽ rất tiện, dễ chia sẻ và hỏi bài trên group. Nhưng ạn chế hỏi group quá nhiều, gặp gì cũng hỏi cứ như vậy bạn sẽ bị phụ thuộc vào người ta và bạn sẽ không tự giải quyết được sau này.

Nếu dùng vscode hay sublime hay công cụ editor nào khác thì nên biết một số extension hỗ trợ để code nhanh hơn cũng như tăng năng suất làm việc như prettier, live server, highlight matching tag, html snippets, emmet, bracket colorizer, babel javascript, auto rename tag, auto complete tag, htmltagwrap…
